

# What Role Am I? 🧑‍💻

An AI-powered career quiz app that helps developers discover which tech role suits them best — based on their personality, work style, and tech preferences.

## 💡 What It Does
Answer 10 fun questions about how you think, work, and code. The AI analyzes your answers and returns your top 3 best-fit tech roles with a match percentage, reasoning, and key skills.

## 🛠️ Tech Stack
- **Frontend**: Blazor WebAssembly (.NET 8)
- **Backend**: ASP.NET Core Web API (.NET 8)
- **AI**: Google Gemini AI / Azure OpenAI
- **Cloud**: Azure App Service, Azure Static Web Apps
- **Other**: CORS, REST APIs, JSON serialization


## ⚙️ Getting Started

### Prerequisites
- .NET 8 SDK
- Visual Studio 2022
- Gemini API key (aistudio.google.com)

### Run Locally
1. Clone the repo
2. Add your Gemini API key to `appsettings.json`
3. Run both projects simultaneously in Visual Studio
4. Navigate to the Blazor app in your browser
